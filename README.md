# Palo Alto Networks Toolkit

Comprehensive toolkit for managing Palo Alto Networks firewalls, Panorama, and Prisma Access.

## Tools

- **Prisma Access** - Egress IP management and API clients
- **Firewall Management** - Configuration and automation scripts
- **GlobalProtect** - Troubleshooting utilities
- **Panorama** - Unused rules analysis
- **Policy Management** - Group address conversion (static to dynamic)

## Requirements

- Python 3.x
- Palo Alto Networks API access
- Required Python packages (see requirements.txt in each tool)

## Usage

Each subdirectory contains specific tools with their own documentation.
